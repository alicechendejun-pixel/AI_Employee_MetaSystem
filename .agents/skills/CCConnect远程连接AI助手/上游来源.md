## 核对结果

官方仓库本身是完整的 CC-Connect 程序，并不是一个只含 `SKILL.md` 的标准 Skill 仓库。官方提供了专门给 Claude Code、Codex、Cursor、Gemini CLI 等 AI 编码 Agent 阅读的 `INSTALL.md`。

本目录因此采用以下方式接入：

1. 把官方安装与配置逻辑整理成标准 `SKILL.md`，供 Codex 项目级发现和调用。
2. 保留中文 Windows 安装配置指南。
3. 提供不含真实密钥的 TOML 配置模板。
4. 不复制、自动运行或编译 CC-Connect 主程序源码。
5. 真正运行时仍通过官方 npm 包、Homebrew 或 GitHub Release 安装 CC-Connect。

## 更新原则

更新本 Skill 前先检查官方仓库的：

- `INSTALL.md`
- `README.zh-CN.md`
- `CHANGELOG.md`
- `config.example.toml`
- Agent 和平台支持列表
- 权限、安全与配置字段变化

禁止在未经审查的情况下自动执行上游安装脚本或覆盖用户本机配置。
