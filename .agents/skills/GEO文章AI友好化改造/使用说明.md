# GEO文章AI友好化改造｜使用说明

## 调用名称

```text
yao-geo-article-friendly
```

## 直接调用示例

```text
使用 yao-geo-article-friendly，把下面这篇公众号文章改造成 GEO 与 AI 搜索友好的版本。
要求保留我的观点，不编造任何数据和引用，输出优化后的全文、改造前后评分、修改说明和待补充来源清单。
```

```text
使用 yao-geo-article-friendly，按 light 模式改造这篇文章，只优化结构、摘要、可引用性和 FAQ，不明显改变我的文风。
```

```text
使用 yao-geo-article-friendly，按 deep 模式改造这篇行业文章。需要补充当前资料的地方先联网核实；无法核实的全部标记为建议补充。
```

## 三种模式

- `light`：轻量改造，适合已经写得比较成熟的个人文章。
- `standard`：默认模式，兼顾结构、证据、语义和可读性。
- `deep`：深度改造，适合官网文章、行业分析、白皮书和需要外部核验的内容。

## 默认交付

1. 改造后的完整文章
2. GEO 优化度评分
3. 改造执行说明
4. 需要补充或确认的内容
5. 风险提示

## 重要限制

- 这是“已有文章改造”Skill，不是从零写稿工具。
- 不承诺 ChatGPT、Gemini、Perplexity 或 Google AI 搜索一定引用或排名。
- 不允许为了提高评分虚构专家引语、报告、数据和案例。
- 当前法律、政策、市场、产品和平台规则必须重新核实。
