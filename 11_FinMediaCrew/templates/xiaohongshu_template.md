# 小红书图文模板

**用于**：每日流量入口。
**目标**：500 字主文 + 9 张图脚本。

---

## 1. 标题（小红书算法核心）

公式：**数字 + 反差 + 利益点**，≤ 20 字。

✅ 示例：
- "3 个 AI 看多同一只股 — 但差 $60，藏着什么"
- "私募风控告诉你：35 岁前必学的 4 个建仓信号"
- "为什么算力股我不追涨 — 用 FinRobot 验证过"

❌ 避免：
- 过长（> 20 字）→ 截断
- 没数字 → 没点击
- 太学术 → 没人懂

---

## 2. 主文（500 字）

### 结构（4 段）

```
【0-100 字】钩子
用 1 个反差 + 1 个真实数字开场。
不要"今天给大家分享"。
直接说"昨天我用 3 个 AI 同时分析了 NVDA — 结果让我意外"。

【100-250 字】铺垫
讲清楚你做了什么事 + 为什么这事重要。
小红书读者不爱抽象，要"我做了 X，发现 Y"。

【250-400 字】3 个洞察
用「1️⃣ 2️⃣ 3️⃣」清单：
1️⃣ {观察 1} → {数据}
2️⃣ {观察 2} → {数据}
3️⃣ {观察 3} → {数据}

【400-500 字】行动建议 + 引导
- 2-3 个具体可做的动作。
- 引导：评论区聊 / 私信免费拿模板。
```

---

## 3. 9 张配图脚本（小红书核心）

| 图位 | 类型 | 标题（图上文字） | 内容描述 |
|---|---|---|---|
| 封面图（图 1） | 数据冲突图 | "3 个 AI 的目标价：$250 / $280 / $310" | 黑底 + 荧光绿三个数字 + 一个大问号 |
| 图 2 | 我做了什么 | "我用 4 个 AI 同时分析了 NVDA" | 4 个 AI 图标排成一排 |
| 图 3 | 第 1 个洞察 | "1️⃣ 三个都看多，但理由不同" | 简洁三栏对比 |
| 图 4 | 第 2 个洞察 | "2️⃣ 关键分歧：数据中心 capex" | 折线图，两条假设线 |
| 图 5 | 第 3 个洞察 | "3️⃣ 风控建议仓位 8% — 不是 30%" | 饼图 |
| 图 6 | 框架 | "我的 4 步建仓法" | 4 步清单 |
| 图 7 | 历史复盘 | "2017 年 AMZN 复盘" | 历史价格图 |
| 图 8 | 一句话总结 | "AI 看多 ≠ 你要梭哈" | 大字句 |
| 图 9 | 引导关注 | "下篇讲 TradingAgents 怎么辩论" | 头像 + 关注按钮设计 |

### 设计规范

- 字体：**思源黑体 Bold** / **PingFang SC Bold**
- 配色：**黑底（#0a0a0a） + 荧光绿（#00ff88） + 白**
- 图比例：**3:4**（小红书最佳）
- 留白：**上下 10% 必须空白**（手机端可读）

## 4. Hashtag（5-8 个）

公式：**1 个泛 + 2 个垂直 + 2 个 SEO + 1-2 个热点**。

✅ 示例：
```
#股票 #NVDA #AI算力 #金融 #投资认知 #私募风控 #女性投资 #2026投资
```

❌ 避免：
- 全用 #股票 #投资 这种泛 tag → 没竞争力
- 全用 #NVDA 这种 SEO tag → 没流量

---

## 5. 评论区话术（必出）

固定 3 条评论模板：

1. 置顶自评：
   ```
   📌 评论区 3 选 1 留言，免费送你《4 个 AI 同时分析 NVDA 的完整对话记录》：
   A. 我也用过 AI 投资
   B. 我是金融新手
   C. 我对私募风控视角感兴趣
   ```

2. 回复套话：
   ```
   ✅ 这条评论我注意到了，可以加个关注我私你 👋
   ```

3. 反对评论的回应（必须有）：
   ```
   你的角度也对 — 这正是 AI 之间的分歧。
   你愿意把它写成一段思考，我转发给我的读者看。
   ```
