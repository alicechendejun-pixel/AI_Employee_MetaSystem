# 漏斗转化优化（5 阶段）

> 金融自媒体的漏斗 = 5 个独立阶段，每个阶段独立 KPI + 独立优化策略。

---

## 1. 整体漏斗

```
┌──────────────────────────────────────────────────┐
│ Stage 1: 曝光        100,000   (CTR 6%)         │
└──────────────┬───────────────────────────────────┘
               ▼
┌──────────────────────────────────────────────────┐
│ Stage 2: 点击 / 观看  6,000    (完读 40%)        │
└──────────────┬───────────────────────────────────┘
               ▼
┌──────────────────────────────────────────────────┐
│ Stage 3: 完读 / 完播  2,400    (关注 5%)         │
└──────────────┬───────────────────────────────────┘
               ▼
┌──────────────────────────────────────────────────┐
│ Stage 4: 关注 / 订阅免费   120  (付费 5%)        │
└──────────────┬───────────────────────────────────┘
               ▼
┌──────────────────────────────────────────────────┐
│ Stage 5: 付费       6  (高客单 8%)               │
└──────────────────────────────────────────────────┘
               ▼
        高客单（1对1 / Cohort）  ~0.5 个
```

**100,000 曝光 → 6 付费 → 0.5 高客单**。

漏斗任何**一阶段下降 50%**，最终付费就掉 50%。所以要**逐阶段优化**。

## 2. Stage 1 → Stage 2：曝光 → 点击（CTR）

### 目标 CTR
- 小红书：**6-8%**（行业平均 3%）
- YouTube：**8-12%**（行业平均 4%）
- 公众号：**5-7%**（行业平均 2%）

### 影响 CTR 的 4 个因子

| 因子 | 权重 | 优化方向 |
|---|---|---|
| 标题 | 50% | A/B 测 3 个标题 |
| 封面 | 35% | 手机端拇指可读 + 数字清晰 |
| 时机 | 10% | 早 8 / 晚 21 |
| 算法识别（标签） | 5% | 5-8 个精准 hashtag |

### 实操：每日"标题/封面 A/B 测试"

每天选 1 个内容**做 3 版标题**，发到 3 个相邻账号 / 时段，测哪个 CTR 最高：

```
版本 A: "为什么 3 个 AI 看多 NVDA 但分歧巨大"
版本 B: "私募风控告诉你：35 岁前必看的 4 个 AI 信号"
版本 C: "我用 4 个 AI 6 分钟做完一份 NVDA 研报"
```

3 个月跑下来 → **30 个标题数据点** → 总结你的"高点击公式"。

## 3. Stage 2 → Stage 3：点击 → 完读 / 完播

### 目标
- 完读率：**40%+**
- 完播率（视频）：**35%+**

### 影响因子

| 时段 | 流失原因 | 优化 |
|---|---|---|
| 0-30s | 钩子不够强 | 重剪开头 5 次 |
| 30s-3 分钟 | 进入正文太慢 | 每 90 秒切换 1 次画面 |
| 3-7 分钟 | 内容密度不够 | 加 1 个真实案例 |
| 7-10 分钟 | 收束太弱 | 重写结尾 + 加金句 |

### 完读率红线

如果某篇内容**完读率 < 25%** → **永久下架**（伤害账号权重）。
如果**完读率 > 50%** → 反向工程，找到爆款公式。

## 4. Stage 3 → Stage 4：完读 → 关注 / 订阅

### 目标关注率
- 公众号：**3%+**（每 100 完读，3 个关注）

- 小红书：**3%+** 同上


#### 动作 3：关注换福利

```

```

## 6. Stage 5 → 高客单：付费 → 1对1 / Cohort / Custom

### 目标高客单转化
- 付费简报 → 1对1：**5-8%**
- 付费简报 → Cohort：**3-5%**
- 付费简报 → Custom：**1-2%**

### 高客单签约的 4 个信号

1. **客户已付费 ≥ 6 个月** —— 信任建立完成。
2. **客户主动问"能不能做 X"** —— 需求被识别。
3. **客户已转发 ≥ 3 次** —— 真实推荐者。
4. **客户在评论区互动 ≥ 5 次** —— 关系建立。

满足 3 条以上 → 主动推高客单。

## 8. 反直觉漏斗规律

### 规律 1：漏斗倒过来看

最赚钱不是**做大顶端曝光**，是**做厚底端高客单**。

- 100 万曝光 + 1% 高客单转化 = 10,000 个高客单（不现实）
- 1 万曝光 + 10% 高客单转化 = 1,000 个高客单（**可达成**）

**所以**：精准 > 流量。

### 规律 2：信任比转化更重要

- 关注后立刻推付费 = 流失 + 关注没了
- 关注后给 30 天免费简报 → 付费转化高 3 倍

**所以**：耐心建立信任。

### 规律 3：少做 > 多做

- 每天 5 篇内容 + CTR 3% < 每周 3 篇内容 + CTR 8%
- 质量 × 频次 = 最终结果

**所以**：宁缺毋滥。

## 10. 一句话总结

> **漏斗 = 信任传递链**。
> 不是每一层都做满，是每一层只送**真心想往下走的人**进入下一层。
> 5% 高质量 > 95% 低质量。
